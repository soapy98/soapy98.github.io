Real Time Graphics C++ Programming and Design

View Code


The aim of the project was the implement a Virtual Gallery in which models had various graphical effects applied onto them. In this it required a well designed and robust application such that the various rendering techniques would run smoothly. We had to also utilise Parasoft, which is a software which goes through code and informs you of bad practises. Some of the rendering techniques are:
1. Gouraud shading
2. Phong shading
3. Bump mapping
4. Displacement mapping
5. Toon shading
6. Transparency
7. Environment mapping of the room
8. Glowing using a post-processing technique
